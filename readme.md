# Scientific Calculator Pro

## Description

Scientific Calculator Pro is a Python-based scientific calculator application built with CustomTkinter.

The project provides a clean graphical user interface with basic and advanced mathematical operations while maintaining a modular software architecture.

## Features

### Basic Operations

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Modulus (%)
- Decimal calculations
- Parentheses support


### Scientific Operations

- Sine (sin)
- Cosine (cos)
- Tangent (tan)
- Logarithm (log)
- Natural logarithm (ln)
- Square root (√)
- Power operations
- Factorial (!)
- Reciprocal (1/x)
- Percentage calculations


### Angle Mode

Supports switching between:

- DEG (Degrees)
- RAD (Radians)


### Memory Functions

Includes calculator memory operations:

- MC - Memory Clear
- MR - Memory Recall
- MS - Memory Store
- M+
- M-


### Additional Features

- Calculation history
- Clear history
- Keyboard-friendly input
- Dark/light theme support
- Responsive calculator layout

## Installation

Clone or download the project.

Install required dependency:
pip install customtkinter



## Running the Application

Navigate to the project directory:
cd ScientificCalculatorPro


Run:
python main.py



## Usage

1. Enter numbers using the keypad or keyboard.
2. Select mathematical operations.
3. Use scientific buttons for advanced calculations.
4. Switch between DEG and RAD depending on the required calculation mode.
5. View previous calculations using history.
